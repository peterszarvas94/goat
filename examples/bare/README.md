# bare

## Goat commands

Run help for goat cli commands:

```bash
goat --help
```

## Project commands

Initialize node for daisyui (only needed first time):

```bash
npm i -C assets/daisyui
```

Start dev server:

```bash
make dev
```

Dump db data:

```bash
make dump
```

Build binary:

```bash
make build
```

Run the binary:

```bash
make run
```
